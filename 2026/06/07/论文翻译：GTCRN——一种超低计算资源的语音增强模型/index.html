论文翻译：GTCRN——一种超低计算资源的语音增强模型

发表于2026-06-07|更新于2026-06-07|论文翻译
|阅读量:

论文信息

原文标题： GTCRN: A Speech Enhancement Model Requiring Ultralow Computational Resources

作者： Xiaobin Rong, Tianchi Sun, Xu Zhang, Yuxiang Hu, Changbao Zhu, Jing Lu

单位： 南京大学现代声学重点实验室；南京大学-地平线智能音频联合实验室；江苏星图信息科技有限公司

会议： ICASSP 2024

代码与音频样例： https://github.com/Xiaobin-Rong/gtcrn

摘要

尽管基于深度学习的语音增强（Speech Enhancement, SE）模型已显著超越传统方法，但它们通常需要大量参数和极高的计算开销，难以在边缘设备上实时部署。本文提出 分组时间卷积循环网络（Grouped Temporal Convolutional Recurrent Network, GTCRN）：在竞争性模型 DPCRN 的基础上，通过分组策略大幅简化网络结构，并引入子带特征提取（SFE）模块与时间循环注意力（TRA）模块以提升性能。最终模型仅含 23.7K 参数，计算量为 39.6 MMACs/s。实验表明，GTCRN 在相近计算负担下显著优于典型轻量模型 RNNoise，并与若干高算力基线模型取得可比性能。

关键词： 语音增强，轻量模型，卷积循环网络

1. 引言

深度神经网络（DNN）的快速发展推动了语音增强领域的重大突破。按处理域划分，DNN 方法可分为时频（T-F）域方法与时域方法。DNN 方法性能优异的同时，往往伴随巨大的模型开销——许多 SOTA 模型需要数 GMAC 到数十 GMAC 的计算量，无法在耳机、助听器等低功耗边缘设备上实用部署。

现有轻量 SE 研究大致有三类思路：

  1. 模型压缩： 对已有高性能模型做剪枝、量化等压缩。
  2. 高效结构设计： 如 TRU-Net 用一维卷积解耦频率/时间轴，并以深度可分离卷积替代标准卷积；Parallel GRU 与优化跳跃连接也可用于构建极小 SE 模型。
  3. 轻量模型 + 后处理： RNNoise、PercepNet 在低分辨率谱包络上做粗增强，再用 pitch comb filter 抑制谐波间噪声；DeepFilterNet 在此基础上用更强的 UNet 增强谱包络并做 deep filtering；DPCRN-CF 则引入可学习 comb filter 提升谐波增强。

然而除 RNNoise 足够紧凑但性能有限外，上述方法对耳机、助听器等场景仍偏大。本文提出 GTCRN，以 DPCRN 为骨干，通过 ERB 滤波器组降低输入冗余、分组卷积与分组 RNN 降低复杂度，并以 SFE 与 TRA 在可控开销下提升性能。模型在 DNS3 与 VCTK-DEMAND 上均显著优于 RNNoise。

2. 分组时间卷积循环网络（GTCRN）

GTCRN 由频带合并（BM）与频带拆分（BS）模块、可选 SFE 模块、编码器、分组双路径 RNN（G-DPRNN）模块与解码器组成，整体结构如图 1 所示。

图 1： GTCRN 整体架构

编码器含 2 个卷积（Conv）块与 3 个分组时间卷积（GT-Conv）块。每个 Conv 块为「卷积层 + BatchNorm + PReLU」，将输入谱映射到高维嵌入并沿频率轴下采样；跳跃连接用于缓解编码阶段信息损失。解码器为编码器的镜像结构，Conv 块替换为反卷积（DeConv）块以恢复原始尺寸；最后一个 DeConv 块使用 tanh 而非 PReLU，将输出约束在 $[-1, 1]$，分别作为估计复数比率掩蔽（CRM）的实部与虚部。

2.1 频带合并与拆分

谱特征可通过 BM 操作下采样，再通过 BS 操作恢复原始分辨率。由于谐波更易出现在低频、极少出现在高频，合并操作仅对 2 kHz 以上 的高频带按 ERB 尺度进行。

2.2 分组双路径 RNN（G-DPRNN）

G-DPRNN 将分组 RNN（GRNN）与双路径 RNN（DPRNN）结合。GRNN 用多组较小 RNN 近似一个大 RNN：输入特征与隐状态各分为 2 组，分别送入参数量为原来一半的 RNN，再经表示重排层得到输出。

DPRNN 原用于一维长序列，也适用于时频特征：帧内 RNN 建模单帧谱模式，帧间 RNN 建模某一频点的时间依赖。帧内采用分组双向 GRU，帧间采用分组单向 GRU，以保证模型因果性。

2.3 分组时间卷积（GT-Conv）

GT-Conv 以 ShuffleNetV2 单元为基础，在深度卷积中引入时间膨胀（temporal dilation），增强长程时间依赖建模能力，结构如图 2 所示。

图 2： 分组时间卷积块

输入沿通道轴一分为二：一支保持不变，另一支依次经过 P-Conv2D、膨胀深度可分离卷积（DD-Conv2D）与 P-Conv2D；两支输出拼接后做 Channel Shuffle 促进信息交换。第二支可选接入 SFE 与 TRA 模块。

2.4 子带特征提取（SFE）

SFE 模块（图 3）增强卷积层对频率信息的利用：先在频率维以核大小 $k$ 做 Unfold，将每个频带与其相邻 $k-1$ 个频带组成子带单元；再 Reshape 将子带单元堆叠到通道维，使原本仅存在于频率维的子带关系进入通道维，便于后续卷积利用。

图 3： 子带特征提取模块

2.5 时间循环注意力（TRA）

TRA 通过乘性注意力掩码对时间特征重标定，结构如图 4 所示。给定输入 $V \in \mathbb{R}^{C \times T \times F}$，先做全局平均池化得到时间能量表示：

$$Z(c, t) = \frac{1}{F}\sum_{f=1}^{F} V^2(c, t, f)$$

其中 $C,T,F$ 分别为通道、时间、频率长度。$Z$ 经 GRU（通道数翻倍）与全连接层（恢复原始通道数），再经 Sigmoid 得到一维注意力掩码，沿频率维复制为二维 T-F 掩码 $A \in \mathbb{R}^{C \times T \times F}$，最终输出：

$$\tilde{V} = V \otimes A$$

其中 $\otimes$ 为逐元素乘法。

图 4： 时间循环注意力模块

2.6 损失函数

损失同时在波形域与谱域计算：

$$\mathcal{L} = \alpha \mathcal{L}{\text{SISNR}}(\tilde{s}, s) + (1-\beta)\mathcal{L}{\text{mag}}(\tilde{S}, S) + \beta\left[\mathcal{L}{\text{real}}(\tilde{S}, S) + \mathcal{L}{\text{imag}}(\tilde{S}, S)\right] \tag{1}$$

其中 $\tilde{s}, s$ 为增强与干净波形，$\tilde{S}, S$ 为对应谱图，$\alpha=0.01$，$\beta=0.3$。各项定义如下：

$$\mathcal{L}{\text{SISNR}} = -\log{10}\frac{|s_t|_2}{|\tilde{s}-s_t|_2}, \quad s_t = \frac{\langle \tilde{s}, s \rangle s}{|s|_2} \tag{2}$$

$$\mathcal{L}_{\text{mag}}(\tilde{S}, S) = \text{MSE}(|\tilde{S}|^{0.3}, |S|^{0.3}) \tag{3}$$

$$\mathcal{L}_{\text{real}}(\tilde{S}, S) = \text{MSE}(\tilde{S}_r/|\tilde{S}|^{0.7}, S_r/|S|^{0.7}) \tag{4}$$

$$\mathcal{L}_{\text{imag}}(\tilde{S}, S) = \text{MSE}(\tilde{S}_i/|\tilde{S}|^{0.7}, S_i/|S|^{0.7}) \tag{5}$$

3. 实验

3.1 数据集

VCTK-DEMAND： 含配对干净/预混合噪声语音。训练集 11,572 条（28 说话人），测试集 824 条（2 说话人），从中取 1,572 条作验证，采样率 16 kHz。

DNS3： 大规模去噪数据集，含多种干净语音、噪声与 RIR；另加入 DiDiSpeech 普通话语料。混合时干净语音先与随机 RIR 卷积，再与随机噪声在 SNR $[-5, 15]$ dB 范围内混合；训练目标保留前 100 ms 反射。共生成 720,000 对 10 秒 noisy-clean 训练样本，验证/测试各 840/800 对，并在 DNS Challenge 3 盲测集上评估，采样率均为 16 kHz。

3.2 实现细节

  • STFT： 32 ms 平方根 Hanning 窗，16 ms 帧移，FFT 长度 512。
  • 输入特征： 带噪谱图实部、虚部与幅度在通道维拼接。
  • BM： 192 个高频带映射为 64 个 ERB 频带，65 个低频带保持不变，得到 129 维压缩特征。
  • SFE： 所有 SFE 模块统一使用 $k=3$。
  • Conv 块： 输出通道 16，核大小 $(1,5)$，步长 $(1,2)$；第二层卷积分组数设为 2。
  • GT-Conv： 三个 DD-Conv2D 层通道数均为 16，核 $(3,3)$，时间膨胀率分别为 1、2、5。
  • 模型规模： 23.7K 参数，39.6 MMACs/s。
  • 训练： Adam，初始学习率 0.001，验证损失连续 5 epoch 不下降则减半。VCTK-DEMAND batch size 4，DNS3 batch size 16；DNS3 训练时将语音切为 8 秒，每 epoch 随机采样 40,000 对。

4. 结论

本文提出 GTCRN，一种仅需 23.7K 参数、39.6 MMACs/s 的语音增强模型。通过对 DPCRN 施加多种轻量化策略，在保持增强性能的同时大幅压缩模型。实验表明，GTCRN 在 VCTK-DEMAND 与 DNS3 上显著优于 RNNoise，并与若干高算力基线模型取得可比性能，适合部署于耳机、助听器等资源受限的边缘设备。
