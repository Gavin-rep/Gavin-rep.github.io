зҹҘиҜҶжҖ»з»“пјҡеЈ°еӯҰеӣһеЈ°ж¶ҲйҷӨпјҲAECпјү

еҸ‘иЎЁдәҺ2023-09-06|жӣҙж–°дәҺ2024-05-05|зҹҘиҜҶжҖ»з»“
|йҳ…иҜ»йҮҸ:

еҹәжң¬дҝЎжҒҜ

вҖӢ еӣһеЈ°е°ұжҳҜжҲ‘д»¬еңЁиҝӣиЎҢе…ЁеҸҢе·ҘйҖҡдҝЎзҡ„иҝҮзЁӢдёӯпјҢиҝңз«Ҝи®ІиҜқиҖ…зҡ„еЈ°йҹіиў«иҝңз«ҜйәҰе…ӢйЈҺйҮҮйӣҶеҲ°йҖҒе…ҘйҖҡдҝЎзҪ‘з»ңпјҢз»ҸиҝҮдј иҫ“еҗҺжқҘеҲ°дәҶиҝ‘з«Ҝжү¬еЈ°еҷЁж’ӯж”ҫпјҢиҝҷдёӘеЈ°йҹіеҸҲиў«иҝ‘з«Ҝжү¬еЈ°еҷЁжҺҘж”¶еҪўжҲҗдәҶеЈ°еӯҰеӣһеЈ°пјҢе№¶дј иҫ“еӣһиҝңз«Ҝжү¬еЈ°еҷЁж’ӯж”ҫпјҢжӯӨж—¶иҝңз«ҜиҜҙиҜқиҖ…е°ұеҗ¬еҲ°дәҶиҮӘе·ұзҡ„еӣһеЈ°гҖӮ

иҖҢжҲ‘д»¬иҝҷйҮҢе°ұжҳҜйҮҮз”ЁдёҖзі»еҲ—з®—жі•жқҘеҺ»йҷӨиҝҷдёӘеӣһеЈ°зҡ„иҝҮзЁӢе°ұеҸ«еҒҡпјҢеЈ°еӯҰеӣһеЈ°ж¶ҲйҷӨ(Acoustic Echo Cancelation), е…¶дё»иҰҒе·ҘдҪңеҺҹзҗҶе°ұжҳҜдёҖдёӘдҝЎеҸ·зҡ„еҜ№ж¶ҲпјҢд№ҹе°ұжҳҜеҰӮдёӢеӣҫжүҖзӨәпјҡ

е…¶еҹәжң¬жҖқжғіе°ұжҳҜпјҡ дј°и®ЎеӣһеЈ°и·Ҝеҫ„зҡ„еҹәжң¬еҸӮж•°пјҢд№ҹе°ұжҳҜе°Ҷж•ҙдёӘеӣһеЈ°зҡ„иҝҮзЁӢи§ҶдёәдёҖдёӘзі»з»ҹпјҢйӮЈд№ҲжӯӨж—¶еӣһеЈ°и·Ҝеҫ„е°ұжҳҜдёҖдёӘи„үеҶІе“Қеә”пјҢд№ӢеҗҺеҫ—еҲ°дёҖдёӘжЁЎжӢҹзҡ„еӣһеЈ°и·Ҝеҫ„пјҢд»ҺиҖҢеҫ—еҲ°жЁЎжӢҹзҡ„еӣһеЈ°дҝЎеҸ·пјҢеҶҚд»ҺжҺҘж”¶дҝЎеҸ·дёӯеҮҸеҺ»иҝҷдёӘеӣһеЈ°дҝЎеҸ·пјҢйӮЈд№Ҳе°ұеҸҜд»Ҙе®һзҺ°еӣһеЈ°жҠөж¶ҲпјҢжүҖд»Ҙе…ій”®е°ұжҳҜжҖҺд№ҲиҺ·еҫ—иҝҷдёӘеӣһеЈ°и·Ҝеҫ„зҡ„и„үеҶІе“Қеә”$\hat{h}(n)$ пјҢ з”ұдәҺжҲ‘д»¬зҡ„еӣһеЈ°и·Ҝеҫ„йғҪжҳҜж—¶еҸҳдё”жңӘзҹҘзҡ„пјҢжүҖд»ҘжҲ‘д»¬зҺ°еңЁдёҖиҲ¬иҖғиҷ‘еј•е…ҘиҮӘйҖӮеә”ж»ӨжіўеҷЁжқҘеҠЁжҖҒи°ғж•ҙжЁЎжӢҹеӣһеЈ°и·Ҝеҫ„пјҢиҮӘйҖӮеә”еӣһеЈ°ж¶ҲйҷӨзҡ„зү№зӮ№е°ұжҳҜиғҪеӨҹиҝӣиЎҢе®һж—¶и·ҹиёӘгҖӮ

иҝҷйҮҢзҡ„$x[n]$ дёәиҝңз«ҜиҜӯйҹіпјҢз»ҸиҝҮжңӘзҹҘзҡ„еЈ°еӯҰи·Ҝеҫ„$w(n)$ еҫ—еҲ°иҝңз«ҜеӣһеЈ°иҜӯйҹі$y(n) = x(n) * w(n)$ , еҶҚеҠ дёҠиҝ‘з«Ҝзҡ„иҜӯйҹі$s(n)$ пјҢйӮЈд№ҲжҲ‘д»¬е°ұеҸҜд»Ҙеҫ—еҲ°жңҹжңӣдҝЎеҸ·$d(n) = y(n) + s(n)$гҖӮ иҖҢжҲ‘д»¬йҖҡиҝҮиҮӘйҖӮеә”ж»ӨжіўеҷЁеҺ»еҫ—еҲ°дёҖдёӘиҝ‘дјјдәҺеӣһеЈ°и·Ҝеҫ„зҡ„е“Қеә”$\hat{w}(n)$пјҢ е°Ҷиҝңз«ҜиҜӯйҹідёҺе…¶иҝӣиЎҢеҚ·з§ҜеҸҜд»Ҙеҫ—еҲ°дёҖдёӘдј°и®Ўзҡ„еӣһеЈ°дҝЎеҸ·$\hat{y}(n)$пјҢ з”ЁжңҹжңӣдҝЎеҸ·еҮҸеҺ»дј°и®Ўзҡ„еӣһеЈ°дҝЎеҸ·еҫ—еҲ°иҜҜе·®дҝЎеҸ·$e(n) = d(n) - \hat{y}(n)$ пјҢиҜҜе·®дҝЎеҸ·зҡ„еҖји¶Ҡе°Ҹдҫҝи¶ҠиҜҙжҳҺиҮӘйҖӮеә”ж»Өжіўз®—жі•жүҖдј°и®Ўзҡ„еӣһеЈ°и·Ҝеҫ„е°ұи¶ҠжҺҘиҝ‘е®һйҷ…зҡ„еӣһеЈ°и·Ҝеҫ„гҖӮ

вҖӢ иҖҢиҮӘйҖӮеә”ж»ӨжіўеҷЁйҮҮз”Ёзү№е®ҡзҡ„иҮӘйҖӮеә”з®—жі•пјҢдёҚеҒңең°и°ғж•ҙжқғеҖјеҗ‘йҮҸпјҢдҪҝдј°и®Ўзҡ„еӣһеЈ°и·Ҝеҫ„$\hat{w}(n)$ йҖҗжёҗи¶Ӣиҝ‘дәҺзңҹе®һзҡ„еӣһеЈ°и·Ҝеҫ„$w(n)$пјҢжүҖд»ҘйҖүжӢ©дёҚеҗҢзҡ„иҮӘйҖӮеә”ж»ӨжіўеҷЁеҸҠз®—жі•еҜ№еӣһеЈ°ж¶ҲйҷӨзҡ„жҖ§иғҪиө·еҲ°жҳҫи‘—еҪұе“ҚгҖӮ

LMS з®—жі• (жңҖе°ҸеқҮж–№з®—жі•)

еңЁд№ӢеүҚзҡ„AECзҡ„еҹәжң¬еҺҹзҗҶдёӯпјҢжҲ‘д»¬дәҶи§ЈдәҶиҜҜе·®дҝЎеҸ·$e(n)$ зӯүдәҺжңҹжңӣдҝЎеҸ·$d(n)$ еҮҸеҺ»ж»ӨжіўеҷЁзҡ„иҫ“еҮәдҝЎеҸ·$\hat{y}$

вҖӢ $e(n) = d(n)-\hat{y} = d(n) - \sum_{i=0}^{N-1}w_i(n)x(n-i) =d(n)- w^T(n)x(n) =d(n)-x^T(n)w(n)$

йӮЈд№ҲжҲ‘д»¬зҡ„зӣ®ж Үе°ұжҳҜдҪҝеҫ—иҜҜе·®дҝЎеҸ·е°ҪеҸҜиғҪзҡ„е°ҸпјҢд№ҹе°ұжҳҜи®©дј°и®Ўзҡ„еӣһеЈ°и·Ҝеҫ„е°ҪеҸҜиғҪең°йҖјиҝ‘еҺҹе§Ӣзҡ„еӣһеЈ°и·Ҝеҫ„пјҢйӮЈд№Ҳе°ұжҳҜи®©иҜҜе·®дҝЎеҸ·$e(n)$ жңҖе°ҸеҢ–пјҢйҮҮз”ЁжңҖе°ҸеқҮж–№иҜҜе·®$(MMSE)$ еҮҶеҲҷжқҘжңҖе°ҸеҢ–зӣ®ж ҮеҮҪж•°:$J(w)$

вҖӢ $J(w) = E[|e(n)|^2] = E[|d(n)-w^T(n)x(n)|^2]$

йӮЈд№Ҳи®Ўз®—зӣ®ж ҮеҮҪж•°$J(w)$ еҜ№$w$ зҡ„еҜјж•°пјҢеҶҚд»ӨеҜјж•°дёә0жңүпјҡ

вҖӢ $\nabla(J(w)) = \frac{\part e^2(n)}{\part w(n)} = -2e(n)x(n)$

йӮЈд№ҲиҝӣдёҖжӯҘзҡ„жҲ‘д»¬зҡ„ж»ӨжіўеҷЁзі»ж•°зҡ„жӣҙж–°е…¬ејҸдҫҝеҸҜд»ҘеҶҷжҲҗпјҡ

вҖӢ $w(n+1) = w(n) - \frac{1}{2}[-\nabla(J(w))] = w(n) + \mu e(n)X(n)$

дёҠйқўзҡ„ејҸдёӯ$\mu$ дёәжӯҘй•ҝеӣ еӯҗпјҢ$\mu$ еҖји¶ҠеӨ§пјҢйӮЈд№Ҳз®—жі•ж”¶ж•ӣеҫ—д№ҹе°ұи¶Ҡеҝ«пјҢдҪҶжҳҜд№ҹдјҡеёҰжқҘжӣҙеӨ§зҡ„зЁіжҖҒиҜҜе·®пјӣ$\mu$ еҖји¶Ҡж…ўпјҢз®—жі•ж”¶ж•ӣи¶Ҡж…ўпјҢдҪҶжҳҜзЁіжҖҒиҜҜе·®д№ҹе°ұи¶Ҡе°ҸгҖӮдёәдәҶдҝқиҜҒз®—жі•зЁіжҖҒж”¶ж•ӣпјҢжүҖд»Ҙ$\mu$ еңЁд»ҘдёӢиҢғеӣҙеҸ–еҖјпјҡ$0 < \mu < \frac{2}{\sum_{i=1}^{N}x(i)^2}$

з»јдёҠжүҖиҝ°пјҢ$LMS$ з®—жі•зҡ„жҜҸж¬Ўиҝӯд»ЈйғҪжңүдёүдёӘжӯҘйӘӨпјҡ

  1. ж»ӨжіўеҷЁиҫ“еҮә$\hat{y}(n)$

  2. дј°и®ЎиҜҜе·®

3. FIR еҗ‘йҮҸзҡ„жҠҪеӨҙжқғеҖјжӣҙж–°пјҢд»ҘеҮҶеӨҮдёӢдёҖж¬Ўиҝӯд»Јпјҡ

е…ідәҺе®һзҺ°йғЁеҲҶзҡ„д»Јз ҒжқҘиҮӘдәҺ$еҮҢйҖҶжҲҳзҡ„еҚҡе®ўпјҡеЈ°еӯҰеӣһеЈ°ж¶ҲйҷӨ(Acoustic Echo Cancellation)еҺҹзҗҶдёҺе®һзҺ° - еҮҢйҖҶжҲҳ - еҚҡе®ўеӣӯ (cnblogs.com)$

йқһеёёж„ҹи°ў

з»“жһңеҰӮдёҠжүҖзӨәпјҢиҝҳжҳҜеҸҜд»ҘзңӢеҲ°з»ҸиҝҮз®—жі•еӨ„зҗҶеҗҺзҡ„дҝЎеҸ·еңЁSTFTеҹҹдёҠе’ҢжҜ”иҫғзӣёдјјгҖӮ

з»јдёҠжүҖиҝ°пјҡLMS з®—жі•зӣёеҜ№жқҘиҜҙжҜ”иҫғз®ҖеҚ•пјҢе®һзҺ°е®№жҳ“пјҢдҪҶжҳҜиө·ж”¶ж•ӣйҖҹеәҰиҫғж…ўпјҢдё”LMSж»ӨжіўеҷЁзі»ж•°жҳҜйҖҗзӮ№иҝӣиЎҢжӣҙж–°зҡ„пјҢе…¶и·ҹиёӘжҖ§иғҪиҫғе·®пјҢдё”жӯҘй•ҝжҒ’е®ҡгҖӮ

з„¶иҖҢдёәдәҶиғҪеӨҹжҸҗй«ҳLMSз®—жі•зҡ„еӨ„зҗҶйҖҹеәҰпјҢеңЁLMSз®—жі•зҡ„еҹәзЎҖдёҠжҸҗеҮәдәҶBlock LMSпјҢдёҺLMSзҡ„йҖҗзӮ№жӣҙж–°зҡ„зү№жҖ§дёҚеҗҢд№ӢеӨ„еңЁдәҺпјҢBLMSз®—жі•жҳҜжҜҸKдёӘйҮҮж ·зӮ№жүҚеҜ№ж»ӨжіўеҷЁзҡ„жқғеҖјиҝӣиЎҢдёҖж¬Ўжӣҙж–°гҖӮ

NLMSпјҲеҪ’дёҖеҢ–еқҮж–№жңҖе°ҸеҢ–пјү

NLMSжҳҜж”№иҝӣиҝҮзҡ„LMSпјҢдё»иҰҒжҖқжғіиҝҳжҳҜе’ҢеҺҹжқҘзҡ„LMSзӣёеҗҢпјҢйӣҶдёӯдәҺйҡҸжңәжўҜеәҰдёӢйҷҚпјҢдҪҶжҳҜе®ғд»¬еңЁжқғйҮҚжӣҙж–°дёҠжңүжүҖдёҚеҗҢпјҢ NLMSз®—жі•дёәдәҶи§ЈеҶіLMSз®—жі•еӣ жӯҘй•ҝжҒ’е®ҡеңЁйҒҮеҲ°дёҚеҗҢе№…еәҰе’ҢеҠЁжҖҒиҢғеӣҙзҡ„дҝЎеҸ·ж—¶пјҢеҸҜиғҪдјҡеёҰжқҘжўҜеәҰеҷӘеЈ°зҡ„ж”ҫеӨ§пјҢж”¶ж•ӣйҖҹеәҰзҡ„дёӢйҷҚзӯүй—®йўҳпјҢе…¶йҖҡиҝҮеј•е…Ҙиҫ“е…ҘдҝЎеҸ·зҡ„е№іж–№иҢғж•°д»ҺиҖҢжқҘеҪ’дёҖеҢ–и°ғж•ҙжӯҘй•ҝпјҢ з”ЁйҡҸзқҖе№…еҖјеҸҜеҸҳзҡ„жӯҘй•ҝеӣ еӯҗжқҘд»Јжӣҝеӣәе®ҡзҡ„жӯҘй•ҝеӣ еӯҗгҖӮ

йҖ’еҪ’жңҖе°ҸдәҢд№ҳжі•(RLS)

йҖ’еҪ’жңҖе°ҸдәҢд№ҳRLSзӣёиҫғдәҺLMSж–№жі•жқҘиҜҙпјҢе…¶жӣҙжіЁйҮҚдәҺе…ЁеұҖдјҳеҢ–пјҢд№ҹе°ұжҳҜе…¶иҜ•еӣҫжүҫеҲ°дёҖз»„жқғйҮҚпјҢдҪҝеҫ—д»Һзі»з»ҹејҖе§ӢиҝҗиЎҢеҲ°еҪ“еүҚж—¶еҲ»пјҢиҫ“еҮәдҝЎеҸ·дёҺжңҹжңӣдҝЎеҸ·зҡ„иҜҜе·®е№іж–№е’ҢжңҖе°ҸпјҢиҝҷйҖҡеёёиҖғиҷ‘жүҖжңүд№ӢеүҚзҡ„ж•°жҚ®жқҘе®һзҺ°пјҲеј•е…ҘдәҶдёҖдёӘйҒ—еҝҳеӣ еӯҗжқҘиЎЎйҮҸд№ӢеүҚзҡ„жҜҸдёӘж•°жҚ®зҡ„жқғйҮҚпјүпјҢжүҖд»Ҙе…¶иў«и®ӨдёәжҳҜе…ЁеұҖзҡ„дјҳеҢ–ж–№жі•гҖӮ

ж•°еӯҰдёҠиЎЁзӨәдёәпјҡ

д№ҹе°ұжҳҜзҺ°еңЁзҡ„зӣ®ж ҮеҮҪж•°еҸҳдёәпјҡ$J_n(w) = \sum_{i=0}^{n}\lambda^{n-i}|e(i)|^2 = \sum_{i=0}^n \lambda^{n-i}|d(i)-w^T(n)x(i)|^2$

иҝҷйҮҢпјҢеҸҜд»ҘзңӢеҲ°жҲ‘д»¬ж»ӨжіўеҷЁзі»ж•°йҖүз”Ёзҡ„жҳҜ$w^T(n)$ пјҢ иҖҢдёҚжҳҜ$w^T(i)$ пјҢ д№ҹе°ұжҳҜиҜҙпјҢеңЁиҮӘйҖӮеә”жӣҙж–°иҝҮзЁӢдёӯпјҢжҲ‘д»¬зҡ„ж»ӨжіўеҷЁжҖ»жҳҜеҸҳеҫ—и¶ҠжқҘи¶ҠеҘҪзҡ„пјҢд№ҹе°ұжҳҜж„Ҹе‘ізқҖеҜ№дәҺд»»дҪ•зҡ„$i < n$ пјҢ $|d(i) - w^T(n)x(i)|$ жҖ»жҳҜжҜ”$|d(i) - w^T(i)x(i)$ иҰҒе°Ҹзҡ„пјҢиҝҷжҳҜеӣ дёәRLSзҡ„дёҖдёӘзү№жҖ§еңЁдәҺе®ғдҪҝз”Ёзҡ„еҪ“еүҚзҡ„жқғйҮҚдј°и®Ў$w(n)^T$ жқҘжңҖе°ҸеҢ–жүҖжңүзҡ„еҺҶеҸІж ·жң¬зҡ„иҜҜе·®з§ҜзҙҜпјҢ еӣ жӯӨеҜ№дәҺд»»дҪ•$i<n$ пјҢ RLSж–№жі•еҫҖеҫҖдјҡдә§з”ҹжӣҙе°Ҹзҡ„зҙҜз§ҜиҜҜе·®пјҢ жҚўеҸҘиҜқжқҘиҜҙпјҢеңЁLMSз®—жі•дёӯпјҢжқғйҮҚжӣҙж–°д»…д»…дҫқиө–дәҺеҪ“еүҚж—¶еҲ»зҡ„иҜҜе·®пјҢиҝҷдёӘз®—жі•жІЎжңүиҖғиҷ‘д»»дҪ•еҺҶеҸІдҝЎжҒҜпјҢиҝҷж„Ҹе‘ізқҖеҰӮжһңеңЁжҹҗдёҖзӮ№дёҠеҸ‘з”ҹдәҶдёҖдәӣзӘҒеҸ‘жҖ§зҡ„еҸҳжҚўжҲ–еҷӘеЈ°е№Іжү°пјҢLMSз®—жі•еҸҜиғҪдјҡиў«иҝҷдәӣе№Іжү°вҖқеј•еҜјвҖңеҲ°дёҖдёӘдёҚеӨӘзҗҶжғізҡ„ж–№еҗ‘пјҢиҖҢRLSеңЁжӣҙж–°жқғйҮҚж—¶пјҢдјҡиҖғиҷ‘жүҖжңүд»ҘеҫҖзҡ„ж ·жң¬пјҢиҝҷж ·е°ұдјҡеёҰжқҘжӣҙеҘҪзҡ„йІҒжЈ’жҖ§гҖӮ

$\lambda$ иў«жҲҗдёәйҒ—еҝҳеӣ еӯҗпјҢеҜ№зҰ»$n$ ж—¶еҲ»и¶Ҡиҝ‘зҡ„иҜҜе·®еҠ иҫғеӨ§зҡ„жқғйҮҚпјҢйҒ—еҝҳе°ұи¶Ҡе°‘пјҢеҜ№дәҺ$n$ ж—¶еҲ»и¶Ҡиҝңзҡ„иҜҜе·®еҠ жҜ”иҫғе°Ҹзҡ„жқғйҮҚпјҢйҒ—еҝҳи¶ҠеӨҡгҖӮ

  • $\lambda = 1$ пјҡ йӮЈд№Ҳд»ЈиЎЁз®—жі•жӯӨж—¶жІЎжңүд»»дҪ•йҒ—еҝҳеҠҹиғҪпјҢйӮЈд№ҲRLSз®—жі•е°ұйҖҖеҢ–жҲҗдёәLMSз®—жі•
  • $\lambda \rightarrow 0$ пјҡ д»ЈиЎЁз®—жі•еҸӘеҜ№еҪ“еүҚзҡ„иҜҜе·®иө·дҪңз”ЁпјҢиҖҢиҝҮеҺ»ж—¶еҲ»зҡ„иҜҜе·®е®Ңе…Ёиў«йҒ—еҝҳгҖӮ

жҲ‘д»¬и®©зӣ®ж ҮеҮҪж•°$J_n(w)$ еҜ№$w $жұӮеҜјпјҢд»Өе…¶жўҜеәҰзӯүдәҺ0пјҢйӮЈд№Ҳеҫ—еҲ°$w$ зҡ„е…¬ејҸеҰӮдёӢпјҡ

йӮЈд№Ҳе°ұеҸҜд»Ҙеҫ—еҲ°$R(n)$е’Ң$r(n)$ зҡ„ж—¶й—ҙйҖ’жҺЁе…¬ејҸпјҡ

дҪҶжҳҜпјҢиҝҷж ·жҲ‘д»¬жҜҸж¬ЎйңҖиҰҒеҜ№$R(n)$ зҡ„жҜҸдёӘзӮ№жұӮе…¶йҖҶзҹ©йҳөпјҢеҫҲдёҚеҲ’з®—пјҢйӮЈд№ҲиҖғиҷ‘дёӢжңүжІЎжңүжӣҙдјҳзҡ„ж–№жі•гҖӮ

д»Ө$P(n) = R^{-1}(n)$ пјҢ $P(n)$ зҡ„ж—¶й—ҙйҖ’жҺЁе…¬ејҸдёәпјҡ

з»јдёҠжүҖиҝ°пјҢжҲ‘д»¬е°ұеҸҜд»Ҙеҫ—еҲ°ж ҮеҮҶзҡ„$RLS$ з®—жі•зҡ„жү§иЎҢжөҒзЁӢеҰӮдёӢпјҡ

  1. еҲқе§ӢеҢ–пјҡ $w(0) =0, p(0) = \delta^{-1}I$, $\delta$ дёәдёҖдёӘеҫҲе°Ҹзҡ„жӯЈж•°пјҢ $I$ дёәеҚ•дҪҚзҹ©йҳөгҖӮ
  2. еҜ№дәҺжҜҸдёҖдёӘж—¶еҲ»$n$ пјҢ йҮҚеӨҚеҰӮдёӢи®Ўз®—пјҡ
    • е…ҲйӘҢиҜҜе·®пјҡ $\epsilon(n) = d(n) - w^T(n-1)x(n)$
    • еўһзӣҠеҗ‘йҮҸпјҡ $k(n) =\frac{P(n-1)x(n)}{\lambda + x^T(n)P(n-1)x(n)}$
    • йҖҶзҹ©йҳөжӣҙж–°пјҡ $P(n) = \frac{1}{\lambda}[P(n-1) - k(n)x^T(n)P(n-1)]$
    • ж»ӨжіўеҷЁжӣҙж–°пјҡ $w(n) = w(n-1) + k(n)\epsilon(n)$
